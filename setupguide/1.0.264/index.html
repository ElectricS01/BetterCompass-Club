BetterCompass, how do I get it?


Dev server

Make sure you have Yarn and Node installed as well as downloading the source code here

1. The first thing you want to do is open the source code folder that should be called BetterCompass-main

2. Now find a hidden file named .env.example and rename it to .env

3. Next you want to locate the proxy folder and open Powershell(Windows) or Terminal(UNIX based) there

4. Then you want to run "yarn"

5. Once you have done that go into the config folder in proxy and replace the config.example.json file with a file named config.json with this data inside it

6. Now run "yarn add sqlite3"

7. After that run "sequelize db:migrate" and "sequelize db:seed" if that fails run "./node_modules/.bin/sequelize db:migrate" and "./node_modules/.bin/sequelize db:seed" instead

8. Finally, run "node ." and go to localhost:23994 in your browser


Now that it's installed, whenever you want to use the Dev server just run "node ." in the proxy folder
